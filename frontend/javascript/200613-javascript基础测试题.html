                                      # 第 12 题

                                      var name = "xiaoming";
var object = {
    name: "chen",
    getNameFunc: function () {
        var that = this;
        return function () {
            return that.name;
        }
    }
}
console.log(object.getNameFunc()());

                                      答：

                                      chen

                                      解析:

                                      object.getNameFunc()先执行，getNameFunc中的this指向object，并赋值给that

                                      ovject.getNameFunc()执行之后，返回函数 function()，产生闭包，可以访问其所在的外层作用域变量that（object）

                                      执行返回的函数(function(){ return object.name })()，最终打印chen
